== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `comment-hierarchy-adjust` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to a comment you want to adjust
4. Change the parent as needed.
5. Enjoy!
